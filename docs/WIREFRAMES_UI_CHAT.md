## 🔄 Estados de Transição

### **Estado: Aguardando Input**
- Campo de input ativo com placeholder
- Botões de preset habilitados
- Painel de resultados vazio
- Sidebar com configurações padrão

### **Estado: Processando**
- Campo de input desabilitado
- Barra de progresso animada
- Botão de cancelar visível
- Logs em tempo real

### **Estado: Sucesso**
- Preview das imagens geradas
- Botões de download ativos
- Opções de reprocessamento
- Notificação de conclusão

### **Estado: Erro**
- Mensagem de erro destacada
- Sugestões de correção
- Botão de tentar novamente
- Link para suporte

## 📐 Especificações Técnicas

### **Breakpoints**
- **Mobile**: 320px - 767px
- **Tablet**: 768px - 1199px  
- **Desktop**: 1200px+

### **Espaçamentos**
- **Padding**: 16px (mobile), 24px (tablet), 32px (desktop)
- **Margem entre elementos**: 8px (mobile), 12px (tablet), 16px (desktop)
- **Altura mínima de cards**: 60px

### **Tipografia**
- **Títulos**: 24px (desktop), 20px (tablet), 18px (mobile)
- **Subtítulos**: 18px (desktop), 16px (tablet), 14px (mobile)
- **Corpo**: 14px (desktop), 13px (tablet), 12px (mobile)
- **Código**: 13px (desktop), 12px (tablet), 11px (mobile)

### **Cores**
- **Primária**: #2c5aa0
- **Secundária**: #667eea
- **Sucesso**: #28a745
- **Aviso**: #ffc107
- **Erro**: #dc3545
- **Fundo**: #f8f9fa
- **Texto**: #333333
- **Bordas**: #e9ecef
